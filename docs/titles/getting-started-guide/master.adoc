= OpenShift.io Getting Started Guide
:author: {rhd} Documentation Team
:email: openshiftio@redhat.com
:revdate: {docdate} {doctime}

This document guides you through your first experiences with Red Hat OpenShift.io. It provides instructions on creating and importing codebases to launch cloud based applications, set up continuous integration and continuous deployment pipelines, analyze and customize the application in a hosted, web-based IDE, and deploy them on OpenShift Online.
